## 💡 Key Concepts

### Mengapa FOL Penting?

**Logika Proposisional**:
```
P: "Socrates mortal"
Q: "Plato mortal"
R: "Aristotle mortal"
... (harus tulis satu-satu)
```

**Logika Predikat**:
```
∀x (Human(x) → Mortal(x))
(Satu formula untuk semua!)
```

### Kekuatan FOL:

- ✅ **Kuantor** - Express "for all" dan "there exists"
- ✅ **Variabel** - Abstract over objects
- ✅ **Predikat** - Properties dan relations
- ✅ **Fungsi** - Operations pada objects
